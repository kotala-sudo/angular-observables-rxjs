Observables & RxJS Example

(Please check the console)

interval - a RxJS operator that creates an observable that emits sequential numbers every specified interval of time
https://rxjs.dev/api/index/function/interval

The interval(1000) emits a number a every 1000 milliseconds i.e. every one second)

pipe - A pipe operator is a function that takes an Observable as its input and returns another Observable. It is a pure operation: the previous Observable stays unmodified.

The pipe operator is being called to take the values emitted by interval and transform them further.

(op1(), op2(), op3(), op4()); this allows you to pipe operations in sequence one after the other. The output of op1( ) becomes the input for op2( ) and so on. In the example pipe( ) function is piping three functions take(), filter() and map in sequence.

take - an operator that emits only the first count values emitted by the source
https://rxjs.dev/api/operators/take

The take(5) operator will only take the first five values emitted in the stream

filter- operator that filters items emitted by the source Observable by only emitting those that satisfy a specified predicate.

filter(x => x%2 == 0) will ensure that it filters only even numbers

https://rxjs.dev/api/operators/filter

map - an operator that applies a given project function to each value emitted by the source Observable, and emits the resulting values as an Observable
https://rxjs.dev/api/operators/map

The map operator is taking the output of the take operator generating the date values in milliseconds the time that elapsed since January 1, 1970 00:00:00 UTC.

subscribe - this function is used to subscribe to the observable returned by the pipe opeator and use the value by logging it onto the console.

You can read more about other RxJS operators here: https://rxjs.dev/guide/operators
